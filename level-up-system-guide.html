🎖️ Complete Level Up System Guide

Everything you need to know about earning XP, leveling up, and unlocking all 100 unique ranks

✨ How the Level Up System Works

🎮 Earning XP

  • Practice Games: 5 XP per game (no reward points)
  • Daily Challenges: 25-50 XP + 5-10 Reward Points
  • Weekly Challenges: 200-700 XP + 50-150 Reward Points

📈 Leveling Up

  • XP formula: 100 × level^1.5
  • Each level unlocks a unique rank symbol
  • Rank updates automatically on level up
  • 100 unique ranks from Stardust to Omniverse

🎁 Rewards

  • Reward Points: Earned from challenges
  • Rank Badges: Display your rank symbol
  • Status: Higher ranks = more prestige

🏆 Challenge Types

📅 Daily Challenges

  • Play Practice Games (3-5 games)
  • Play Competition Games (1-2 games)
  • Score Threshold (800-1200+ points)
  • Game Marathon (4-6 total games)

Rewards: 25-50 XP + 5-10 Reward Points

📆 Weekly Challenges

  • Weekly Practice Champion (15-25 games)
  • Weekly Competitor (10-15 games)
  • XP Collector (100-200 XP)
  • Level Up Legend (1-3 levels)

Rewards: 200-700 XP + 50-150 Reward Points

🎖️ All 100 Rank Symbols

Every level unlocks a unique rank with a special Unicode symbol. From the humble Stardust (◉) at Level 1 to the ultimate Omniverse (♌) at Level 100, each rank represents your cosmic journey through the game.
