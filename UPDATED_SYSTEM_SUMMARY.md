## 🎯 Regular Listings Base Price System

### 📋 Listing Management Service:
- **Base Price Thresholds**: Each listing requires its base price in coins to start timer
- **Auto-Transition**: Listings automatically move to countdown when threshold is met
- **Progress Tracking**: Real-time progress bars showing coins collected vs. base price
- **Entry Management**: Users can make 1-3 entries per listing (max 3 total)

### 🔄 How Listings Work:
1. **Waiting Phase**: Listing waits for base price to be met (e.g., $1099 for iPhone)
2. **Threshold Met**: When $1099 in coins collected, timer automatically starts
3. **Countdown Active**: 24-hour (or custom) countdown begins
4. **Competition Phase**: All participants can now play their mystery games
5. **Winner Selection**: Highest score wins the item

### 📊 Example Listing Flow:
```
iPhone 15 Pro ($1099 base price)
├── Phase 1: Collecting coins (0/1099)
├── Users enter with 1-3 coins each
├── Progress: 500/1099 coins (45%)
├── Progress: 800/1099 coins (73%)
├── Progress: 1099/1099 coins (100%) ✅
├── Phase 2: Timer starts automatically (24 hours)
├── All participants can now play games
└── Winner determined by highest score
```
